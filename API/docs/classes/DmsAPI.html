VerifyCredential()

VerifyCredential(string $sUsername, string $sPassword, string $sUserAPIKey, string $fConfig) : string

This Method Authenticates a user to the doxbox DMS. DoxBox User and Password Authentication is the only authentication method that is supported at this time.

Parameters

string $sUsername

Username to authenticate.
string $sPassword

MD5 Hash of the Users Password.
string $sUserAPIKey

API Key generated using ./genapikey.php .
string $fConfig

Optional location of the api config file that contains the Authorized API Keys.
Default Location: ./config/api.config.php.

Throws

array

$aError On error Contains an error number and associated error message.

  Array
  (
     [code] => 0010
     [msg] => You API KEY is Missing Access Denied
  )
array

$aSession Containing a valild Session ID to be used in subsequent Methods.

  Array
  (
     [sessid] => 3f0265c2f54dc5802fb557af55e3287b
  )

Returns

string —

The thrown array is parsed by the 'Response' Method into a JSON or XML

            string, depending on ResponseType Requested by the Server. Then Sent back
            to the client to be parsed and appropiate action taken by the client.
